Simple Features

I implemented the following simple features in Nori:

  • UV Mapping
  • Spherical Mapping
  • Transparent Textures
  • Texture Superposition
  • Normal Maps
  • Bump Maps
  • Rough Conductors
  • Bridge Modelling

The relevant header files are in /include/nori/texture.h for all the different textures, /include/nori/texturemappings.h for the UV and Spherical mappings, /include/nori/normalmaps.h and /include/nori/bumpmaps.h for the Normal and Bump maps. Some convenience adjustments have been made in /include/nori/bsdf.h for all the different texture types.

UV Texture Mapping

Textures are a must have to make a scene alive. I implemented simple UV texture mapping, a constant color convenience texture and also a procedural checkerboard texture to have some nice looking testing surface.
The source files of interest are the following:

  • /src/imagetexture.cpp
  • /src/constant.cpp
  • /src/checkerboard.cpp
